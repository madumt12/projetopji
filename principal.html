Bem-vindo ao EcoPlay!

Objetivo do Site

Este site foi criado com o objetivo de ser uma ferramenta educacional completa para ajudar estudantes a aprenderem mais sobre os diferentes biomas do nosso planeta. Aqui, você encontrará lições interativas, perfis detalhados de cada bioma, além de recursos adicionais para aprofundar seus conhecimentos.

Nossa Missão

Nossa missão é tornar o aprendizado sobre biomas mais acessível e interessante para todos. Acreditamos que entender os biomas é essencial para a conscientização ambiental e para a preservação do nosso planeta.

Quem Somos

Somos três estudantes do curso técnico de Informática para Internet do Instituto Federal de São Paulo (IFSP) - Campus Salto. Este site foi desenvolvido como parte de um projeto para a matéria Projeto Integrador. Nós combinamos nossas habilidades em HTML e CSS para criar uma plataforma que não só é funcional, mas também visualmente atraente e fácil de usar.

Esperamos que você aproveite e aprenda muito com o nosso site!
